Cryptocoin Ticker Images

This is a simple app based on the idea of the Bitcoin Tickers app. Given a special URL, the app will generate images representing the value of a particular coin.

For example, to generate an image for the current highest bid price for DVC in BTC on Vircurex, use the following URL:
http://dvcticker.appspot.com/vircurex/dvc/btc.png
The resulting image:

The only currently supported exchange is Vircurex (more coming soon). Supported currencies are: BTC, USD, EUR, DVC, IXC, LTC, NMC, PPC, TRC.
